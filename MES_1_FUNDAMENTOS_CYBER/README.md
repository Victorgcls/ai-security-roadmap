## Principais Conceitos

- **Tríade CIA:** Confidencialidade, Integridade, Disponibilidade
- **CVE/CWE/CVSS:** Sistema de classificação de vulnerabilidades
- **Fases do Hacking Ético:** Reconhecimento → Escaneamento → Exploração → Relatório
- **Frameworks:** NIST, ISO 27001, MITRE ATT&CK, CIS Controls
- **Regulamentação:** LGPD, gestão de riscos
