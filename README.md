## Por que IA + Security?

Pouquíssimos profissionais combinam IA e Cybersecurity. Enquanto modelos de ML se tornam infraestrutura crítica, a demanda por quem sabe protegê-los cresce mais rápido do que a oferta. Este repositório documenta minha jornada construindo exatamente esse perfil.
